# Kingdom Defense V1

[▶ Play Kingdom Defense](https://themanthatishappy.github.io/Kingdom-Defense-V1/)

If the link shows a 404, enable GitHub Pages for this repo:

1. Open **Settings** in this repository.
2. Click **Pages** on the left.
3. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
4. Set **Branch** to `main` and folder to `/ (root)`.
5. Click **Save**.
